## 基础用法
该组件需配合`el-form`使用
::: tip
基于`el-select`做了一些增强:
-  `clearable`, `filterable`, `defaultFirstOption`这三个属性修改为默认为true
-  options属性支持传入数组和对象,传入数组时,可同步传入`optionsAttr`属性,该属性为一个字符串用逗号分隔,为label,key的别名,传入对象时则传入简单key-value结构

:::
